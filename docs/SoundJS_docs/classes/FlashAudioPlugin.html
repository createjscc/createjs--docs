FlashAudioPlugin 类

扩展： AbstractPlugin
定义: FlashAudioPlugin:41
模块: SoundJS

播放声音使用的Flash实例。这个插件是默认不使用，且必须手动进行注册 Sound 使用 registerPlugins 方法。 建议使用此插件，如果旧的浏览器，如IE 8需要完善的支持，被列入。

此插件需要FlashAudioPlugin.swf和swfObject.js，它编译成缩小的FlashAudioPlugin-X.X.X.min.js文件。您必须确保 swfPath 使用这个插件的时候，这样的脚本可以找到SWF设置。

实例

 createjs.FlashAudioPlugin.swfPath = "../src/soundjs/flashaudio";
 createjs.Sound.registerPlugins([createjs.WebAudioPlugin, createjs.HTMLAudioPlugin, createjs.FlashAudioPlugin]);
 // 再添FlashAudioPlugin作为后备，如果WebAudio和HTMLAudio不起作用。

注意，SWF被嵌入到一个容器DIV（用“SoundJSFlashContainer”的ID和类名），以及将具有“flashAudioContainer”的标识。容器div被定位为1像素屏幕外的左侧，以避免显示1x1像素的白色正方形。

对Flash音频已知的浏览器和操作系统的问题

所有浏览器

  • 可以有flash播放器开始音频播放的延迟。这一直是最引人注目的Firefox浏览器。 Unfortunely这与Flash播放器和浏览器的问题，因此不能由SoundJS加以解决。
