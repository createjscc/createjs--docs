CSSPlugin类

定义于: CSSPlugin:39
模块: TweenJS

一个处理数字CSSTweenJS插件 字符串属性(ex. top, left)。TweenJS加载后使用简单的安装:

 createjs.CSSPlugin.install();

你可以调整CSS属性将通过修改 cssSuffixMap 属性. 目前, 这个top, left, bottom, right, width, height 有"px"后缀.

需要注意的是：TweenJS压缩文件中并不包含CSSPlugin文件（需要自行引入）。

cssSuffixMap

对象 静态的

定义于 cssSuffixMap:58

定义的默认后缀地图CSS渐变类，这可以覆盖在每个渐变的基础上通过指定一个个人渐变cssSuffixMap值， 对象映射的CSS属性名称的后缀时使用阅读或设置这些属性。例如地图形式{top:"px"}指定渐变"top"CSS属性时,应该使用"px"后缀(例如target.style="20.5 px")。 这只适用于Tween类"css"配置属性设置为true .
