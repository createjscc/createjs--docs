ButtonHelper Class

定义: ButtonHelper:41
模块: EaselJS

这个buttonhelper是一个辅助类来创建交互式按钮MovieClip 或者 Sprite 实例。 该类将从对象中拦截鼠标事件，并自动调用gotoAndStop 或者 gotoAndPlay, 要为各自的动画标签，添加一个指针光标，并允许用户定义一个打状态帧。

这个buttonhelper实例不需要增加的阶段，但是参考应防止垃圾收集。

请注意，除非你打电话，否则就不工作enableMouseOver.

实例

 var helper = new createjs.ButtonHelper(myInstance, "out", "over", "down", false, myInstance, "hit");
 myInstance.addEventListener("click", handleClick);
 function handleClick(event) {
     // 点击触发
 }

定义 ButtonHelper:41

参数:

  • target Sprite | MovieClip

    管理实例。

  • [outLabel="out"] String optional

    当用户从按钮中滚动时，该标签或动画会继续使用。

  • [overLabel="over"] String optional

    当用户在按钮上滚动时，标签或动画会继续使用。

  • [downLabel="down"] String optional

    当用户按下按钮时，标签或动画会继续进行。

  • [play=false] Boolean optional

    如果辅助应该叫“gotoAndPlay”或“按钮改变状态时gotoAndStop”。

  • [hitArea] DisplayObject optional

    一个可选的项目，以使用该按钮的点击状态。如果没有定义，则按钮的可见状态将被用来代替。注意实例作为“目标”的论点，可用于hitstate相同。

  • [hitLabel] String optional

    在定义hitarea界hitarea实例的标签或动画。如果这是空的，然后hitarea默认状态将被使用。*
