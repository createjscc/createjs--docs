Event类

定义于: Event:48
模块: CreateJS

包含属性和方法为使用EventDispatcher共享的所有事件。

注意,事件对象往往重用,所以你不能依赖于超出调用堆栈时接收到的事件对象的状态。

currentTarget

Object 只读的

定义于 currentTarget:80

当前目标冒泡事件被派遣。对于non-bubbling事件,总是一样的目标。 例如,如果childObj.parent = parentObj,将会从childObj生成一个冒泡事件 然后parentObj侦听器将接收的事件target=childObj (最初的目标)和currentTarget = parentObj(侦听器添加)。

默认: null
