SamplePlugin类

定义于: SamplePlugin:39
模块: TweenJS

一个示例TweenJS插件。 这个插件实际上并没有以任何方式影响渐变动画,这仅仅是为了文档如何构建TweenJS插件。 请看内联注释的代码。

TweenJS插件只是一个对象,公开一个属性(优先)和三种方法(init, step, and tween)。 通常一个插件也会公开install的方法,虽然这并不是必需的。

定义于 init:72

由TweenJS调用初始化一个新的渐变属性时,这个插件被注册。通常,调用 Plugin.init 会立即紧随其后的是一个 Plugin.step的调用。

参数:

  • tween Tween

    渐变的相关实例。

  • prop 字符型

    被初始化的属性名称。

  • value Any

    这个渐变的目标得属性的当前值。

返回:

任何:

开始渐变这个属性的值。 在大多数情况下,你只会简单的返回参数值,但有些插件可能需要修改的起始值。

当添加一个新的包括属性的渐变时这个插件将被TweenJS调用（ie.用新的"to"行为添加一个渐变）。

参数:

  • tween Tween

    渐变的相关实例。

  • prop 字符型

    被初始化的属性名称。

  • startValue Any

    这个渐变的目标得属性的当前值。在开始的一步，属性的值。如果这是第一步，这将是和init值一样的,或者如果不是endValue前一步还是一样的 。

  • injectProps 对象

    一个插件可以添加其他的通用对象属性,应该更新这个步骤。

  • endValue Any

    属性的值的最后一步。

所有的 静态的

定义于 tween:107

时调用一个渐变属性提出这个插件注册。

参数:

  • tween Tween

    渐变的相关实例。

  • prop 字符型

    被渐变的属性名称。

  • value Any

    当前渐变的属性值,由TweenJS计算。

  • startValues 对象

    当前这步开始的所有属性值的一个散列。 你可以用startValues[道具]访问当前属性的开始值。

  • endValues 对象

    当前这步结束的所有属性值的一个散列。

  • ratio 数值型

    表明了通过当前步骤进展的一个值。 这数字一般在0和1之间,虽然一些缓解将生成这个范围外的值 。

  • wait 布尔型

    示当前步骤是否是一个“wait(等待)”的一步。

  • end 布尔型

    表示渐变是否走到尽头了

返回:

任何的:

返回的值应该被分配到目标属性。例如 返回 Math.round(value) 将指定默认的计算的值是一个整数。返回渐变。 忽略将防止分配到目标属性一个值的渐变。

未知的 静态的

定义于 priority:53

TweenJS用来决定何时调用这个插件。 插件有更高的优先级的方法在较低的优先级插件之前调用。优先级值可以是任何积极或消极的数字。
