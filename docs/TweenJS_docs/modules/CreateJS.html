CreateJS模块

定义于: CreateJS:36

贯穿所有CreateJS共享库的类的集合。类中包含每个库的压缩文件和可直接createsjs命名空间。

Example

 myObject.addEventListener("change", createjs.proxy(myMethod, scope));

这个模块提供了以下类:

  • Event
  • EventDispatcher
  • Ticker
  • Utility Methods
